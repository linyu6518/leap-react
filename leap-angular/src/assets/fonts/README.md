# Graphik Font Files

请将以下 Graphik 字体文件放置在此目录中：

- `Graphik-Light.woff2` (font-weight: 300)
- `Graphik-Light.woff` (font-weight: 300)
- `Graphik-Regular.woff2` (font-weight: 400)
- `Graphik-Regular.woff` (font-weight: 400)
- `Graphik-Medium.woff2` (font-weight: 500)
- `Graphik-Medium.woff` (font-weight: 500)

如果您有 Graphik 字体的许可证，请从您的字体提供商下载这些文件并放置在此目录中。

如果没有 Graphik 字体文件，系统将回退到系统默认字体。
